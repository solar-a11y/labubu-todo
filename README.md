# 라부부 할 일 목록 앱

라부부 캐릭터와 함께하는 모바일 친화적 할 일 관리 앱입니다.

## 기능

- **작업 추가** — 제목과 선택적 마감일을 입력 후 추가 버튼 또는 Enter
- **완료 표시** — 체크박스 클릭으로 완료/미완료 토글
- **작업 삭제** — 각 항목 오른쪽 × 버튼으로 개별 삭제
- **완료 항목 일괄 삭제** — 완료된 항목이 있을 때 나타나는 "완료 삭제" 버튼
- **필터** — 전체 / 진행 중 / 완료 탭으로 목록 전환
- **마감일 강조** — 3일 미만 남은 항목은 빨간색, 당일 마감은 주황색, 만료는 취소선 처리
- **카운터** — 전체 / 남은 항목 / 완료 개수를 상단에 실시간 표시
- **데이터 유지** — `localStorage`에 저장되어 새로고침 후에도 작업 목록 보존

## 사용 방법

별도 설치 없이 `todo.html` 파일을 브라우저에서 열면 바로 실행됩니다.

## 파일 구조

단일 HTML 파일(`todo.html`)로 구성되며, CSS와 JavaScript가 모두 인라인으로 포함되어 있습니다.

| 구성 요소 | 역할 |
|---|---|
| CSS 변수 (`--accent`, `--urgent` 등) | 전체 색상 테마 관리 |
| `dueBadge()` | 마감일 계산 및 강조 배지 생성 |
| `taskHTML()` | 개별 작업 카드 HTML 생성 |
| `render()` | 카운터 업데이트 + 필터링 + 목록 렌더링 |
| `localStorage` | 작업 데이터 영구 저장 |
